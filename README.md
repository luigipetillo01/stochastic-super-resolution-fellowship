# Stochastic Super Resolution in 2D-Lattices
### Research Project | AURORA Excellence Fellowship 2025

This repository contains the final presentation for my 3weeks-research internship at the **Universität Innsbruck** (Leopold-Franzens-Universität) within the **Ferlaino Group**.

## Project Summary
The project focused on a classical **Monte-Carlo simulation** to study the imaging of atoms in a 2D-lattice. We explored stochastic methods to overcome standard resolution limits in quantum gas experiments.
